# XYZ-Electronics---Sales-Excellence-Dashboard-2022


Project Overview :
This project was designed for XYZ Electronics to recognize the top-performing sales employees across three divisions: Dhaka, Chittagong, and Sylhet. The primary goal was to identify the top 3 employees eligible for the iPhone 16 Pro Max reward based on their 2022 sales performance.

Key Insights :
Top Performers: The dashboard identifies the regional heroes like Naseef (Sylhet), Anika (Chittagong), and Mushfiq (Dhaka).
Product Performance: Laptops (36.36%) and Tablets (35.54%) were the highest-selling products.
Regional Success: Chittagong emerged as the leading division in terms of total sales volume.

Tools Used :
Power BI and Excel .

Data Visualization: 
Bar charts, Pie charts, and Date Slicers.
